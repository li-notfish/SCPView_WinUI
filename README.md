# SCPView_WinUI
一个应用程序，用于查看SCP项目。目前只有查看SCP项目内容，SCP故事以及其他离线功能后面再完善。
# 项目结构
``````
├─SCPView_WinUI
│  ├─SCPView_WinUI //WinUI项目
│  │  ├─Helper //帮助类文件夹
│  │  ├─Pages //页面文件夹
│  │  ├─Services //服务类文件夹
│  │  └─ViewModels //视图模型文件夹
│  └─SCPView_WinUI (Package) //打包的WinUI项目
│      └─Images //项目图片文件夹
├─SCPView_WinUI.Data //数据项目
│  ├─Model //模型文件夹
│  ├─Parser //网页解析文件夹
│  └─Utils //工具文件
└─SCPView_WinUI.Test //测试工程项目
``````
